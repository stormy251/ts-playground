# Deno Project

A modern Deno project with TypeScript support and VS Code configuration.

## Setup

1. Install Deno by following the
   [official instructions](https://deno.land/#installation)
2. Install the
   [Deno VS Code extension](https://marketplace.visualstudio.com/items?itemName=denoland.vscode-deno)
3. Open this project in VS Code

## Features

- TypeScript support with type checking
- Automatic formatting on save
- Linting on save
- Predefined tasks for development

## Available Commands

- `deno task start` - Run the application
- `deno task dev` - Run with watch mode (auto-reload on changes)
- `deno task test` - Run tests
- `deno task lint` - Run linter
- `deno task fmt` - Format code

## VS Code Configuration

This project includes VS Code settings that:

- Enable the Deno extension
- Configure formatting on save
- Enable linting
- Set up code actions on save

For more information on Deno and VS Code, see the
[official documentation](https://docs.deno.com/runtime/reference/vscode/).
